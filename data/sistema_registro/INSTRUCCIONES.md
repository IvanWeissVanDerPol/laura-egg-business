# Sistema de Registro - Granja Cabral

## Cómo registrar por WhatsApp

Laura solo necesita enviar mensajes al grupo con este formato simple:

### Para registrar PRODUCCIÓN del día:
```
Producción:
A 84
B 104
C 30
S 19
J 3
picado 2
```

### Para registrar una VENTA:
```
Venta: [cliente] [cantidad] [tipo] [pagado/debe]
```
Ejemplos:
- `Venta: Fada 20 A pagado`
- `Venta: Dalila 15 B debe`
- `Venta: Coti 10 S 5 A pagado`

### Para registrar un GASTO extra:
```
Gasto: [concepto] [monto]
```
Ejemplo: `Gasto: reparación galpón 500000`

### Para registrar un PAGO de deuda:
```
Pago: [cliente] [monto]
```
Ejemplo: `Pago: Dalila 500000`

### Para consultar:
- "¿Cuánto stock tengo?" → stock actual
- "¿Cuánto vendí hoy?" → ventas del día
- "¿Quién me debe?" → deudas pendientes
- "¿Cuánto gané esta semana?" → resumen semanal
- "Resumen del mes" → resumen mensual
