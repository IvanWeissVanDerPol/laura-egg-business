## 3. Delivery Operations

### Pre-Delivery Checklist

**Day Before:**
- [ ] Confirm all orders via WhatsApp
- [ ] Calculate total volume needed
- [ ] Check vehicle condition
- [ ] Verify cooler temperature

**Morning of Delivery:**
- [ ] Load eggs by route (Zone A first = furthest)
- [ ] Check egg condition before loading
- [ ] Verify packaging quantities
- [ ] Load invoices/receipts
- [ ] Prepare collection envelopes
- [ ] Check fuel level
- [ ] Vehicle inspection

### Loading Order

Load in reverse order (last stops first):
1. Zone A/B supplies (top/back)
2. Zone C supplies
3. Zone D supplies (front/easy access)

### Delivery Procedure

```
ARRIVAL:
1. Greet customer
2. Verify order
3. Check product condition
4. Load eggs

PAYMENT:
5. Present invoice
6. Collect payment
7. Issue receipt
8. Note any issues

DOCUMENTATION:
9. Get signature
10. Update delivery log
11. Note customer feedback
12. Next delivery scheduling
```

### Post-Delivery

- [ ] Return to farm by scheduled time
- [ ] Unload any returns
- [ ] Record all deliveries
- [ ] Bank/safe cash receipts
- [ ] Clean vehicle
- [ ] Note any problems
- [ ] Report to manager

| Issue | Action | Timeline |
|-------|--------|----------|

### Route Efficiency Tips

1. **Consolidate orders** - Multiple small orders in same area
2. **Early morning** - Cooler, faster traffic
3. **Optimize sequence** - Minimize backtracking
4. **Batch deliveries** - Same-day zones together
5. **Regular customers** - Predictable quantities
