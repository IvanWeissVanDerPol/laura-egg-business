# ⚡ TRANSFORMACIÓN: Biogás
## Plan de Negocio - Granja Cabral
> **Última actualización:** Marzo 2026

---

## 📋 DESCRIPCIÓN DEL PROCESO
- **Producto:** Biogás (metano) + Biol (fertilizante líquido)
- **Proceso:** Digestión anaeróbica de residuos orgánicos
- **Aplicación:** Energía para la granja + Fertilizante

---

## 📊 PRINCIPIO DE FUNCIONAMIENTO
```
RESIDUOS ORGÁNICOS
     │
     ▼
DESTRUCCIÓN ANAERÓBICA
     │
     ├──→ 60-70% METANO (CH₄) ──→ BioGas
     │         │
     │         └──→ Cocina, iluminación,发电机
     │
     └──→ 30-40% CO₂ ──→ Emisiones
          +
          └→ Residuo líquido ──→ BIOL (fertilizante)
```

## ⚡ USOS DEL BIOGÁS
1. **Cocina** — Estufa de biogas (reemplaza gas garrafa)
2. **Calentamiento de agua** — Para galpones (clima frío)
3. **Generación eléctrica** — Con microturbina (opcional)
4. **Iluminación** — Lámparas de biogás

## ⚠️ DESAFÍOS
- **Inversión significativa** — Requiere escala
- **Mantenimiento técnico** — Conocimiento necesario
- **Clima frío** — Digestión más lenta en invierno
- **Producción constante** — Necesita residuos diarios
