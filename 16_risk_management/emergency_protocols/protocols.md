## 3. Disease Outbreak Protocol

### Warning Signs

| Symptom | Threshold | Action |
|---------|-----------|--------|
| Sudden death | >5 birds/day | Immediate |
| Respiratory signs | Any | Within 24h |
| Drop in production | >15% | Within 24h |
| Water consumption drop | >20% | Within 24h |
| Unusual droppings | Any | Observe |
| Egg abnormalities | >10% | Monitor |

### Response Protocol

**Step 1: Identification (Day 1)**
```
1. Observe and document symptoms
2. Isolate affected birds if possible
3. Call veterinarian immediately
4. Do NOT move birds
5. Do NOT allow visitors
6. Begin biosecurity lockdown
```

**Step 2: Assessment (Day 1-2)**
```
1. Vet visit and diagnosis
2. Sample collection if needed
3. Test results interpretation
4. Mortality assessment
5. Production impact calculation
```

**Step 3: Containment (Day 2-3)**
```
1. Implement quarantine
2. Increase biosecurity
3. Foot baths at maximum
4. No new birds
5. Equipment isolation
6. Staff restriction
```

**Step 4: Recovery Plan (Day 3+)**
```
1. Follow vet treatment plan
2. Enhanced monitoring
3. Vaccination if applicable
4. Plan for restocking
5. Customer communication
6. Documentation for insurance
```

### Disease Outbreak Decision Matrix

| Severity | Mortality | Production Loss | Action |
|----------|-----------|-----------------|--------|
| Low | <5 birds | <10% | Treat, monitor |
| Medium | 5-20 birds | 10-25% | Vet treatment, isolate |
| High | 20-100 birds | 25-50% | Cull affected, full quarantine |
| Critical | >100 birds | >50% | Cull flock, deep clean, restart |

## 4. Fire Emergency Protocol

### Prevention
- [ ] No open flames near facilities
- [ ] Electrical system checked annually
- [ ] Fire extinguisher accessible
- [ ] Clear vegetation around buildings
- [ ] No smoking near barns
- [ ] Electrical overload protection

### Fire Response

**If Fire Starts:**
```
1. SOUND ALARM - shout "FUEGO"
2. CALL 132 immediately
3. Evacuate all people first
4. If small and safe: Use extinguisher
5. If large or unsafe: EVACUATE
6. Meet at designated assembly point
7. Account for all people
8. Direct fire department on arrival
```

## 5. Natural Disaster Protocol

### Storm/Flood Preparation

**When Alert Issued:**
```
1. Secure loose items
2. Check drainage
3. Move birds to high ground if flooding risk
4. Backup power source ready
5. Emergency supplies prepared
6. Communication plan ready
7. Insurance documents accessible
```

### During Storm
```
1. Monitor weather continuously
2. Stay indoors
3. If flooding: Move to highest point
4. Monitor bird welfare
5. Document any damage
```

### After Storm
```
1. Wait for official all-clear
2. Inspect all facilities
3. Check for damage
4. Assess bird condition
5. Document losses for insurance
6. Begin repairs
7. Customer notification if delivery affected
```

## 6. Equipment Failure Protocol

### Critical Equipment Failures

| Equipment | Risk Level | Backup Plan |
|-----------|------------|-------------|
| Water pump | HIGH | Reserve tank, manual filling |
| Feed system | MEDIUM | Manual feeding |
| Egg cooler | HIGH | Ice packs, shade, faster delivery |
| Ventilation | CRITICAL | Open windows/doors, reduce density |
| Power supply | MEDIUM | Generator, battery backup |

### Response by Equipment

**Water Pump Failure:**
```
1. Switch to backup pump if available
2. Use reserve water tank
3. Manual watering if needed
4. Call repair immediately
5. Monitor birds closely
Timeline: 2-4 hours max without water
```

**Cooler Failure:**
```
1. Move eggs to shade
2. Use ice packs around eggs
3. Prioritize delivery
4. Sell as "same-day" or process
5. Call repair immediately
Timeline: 4-6 hours before quality impact
```

**Power Outage:**
```
1. Activate generator
2. Manual ventilation if needed
3. Reduce flock density
4. Monitor temperature
5. Prioritize egg cooling
Timeline: Depends on temperature
```

## 7. Security Emergency

### Theft Response
```
1. Do NOT confront intruders
2. Call Police: 911 immediately
3. Note description if visible
4. Secure remaining inventory
5. Document losses
6. File police report
7. Notify insurance
8. Review and improve security
```

### Predator Attack
```
1. Assess situation - birds at risk?
2. Remove birds to safety
3. Set traps or call pest control
4. Repair housing/penetration points
5. Monitor for wounded birds
6. Consider vaccination
7. Document losses
```

## 8. Personnel Emergency

### Worker Injury
```
1. Ensure scene is safe
2. Call ambulance if serious: 141
3. Provide first aid if trained
4. Contact family/emergency contact
5. Document incident
6. Review safety procedures
7. Report to authorities if required
```

### Sudden Worker Absence
```
1. Contact backup worker
2. Owner covers if needed
3. Simplify tasks
4. Prioritize critical operations:
   - Feeding
   - Watering
   - Egg collection
5. Reduce flock temporarily if needed
```
