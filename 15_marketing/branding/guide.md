# Brand Development Guide

## 1. Brand Identity

### Brand Name Options

| Option | Description | Pros | Cons |
|--------|-------------|------|------|
| **Granja Cabral** | Farm name (recommended) | Personal, memorable | Less creative |
| **Huevos del Campo** | Country eggs | Natural, appealing | Generic |
| **La Granja de la Route 2** | Location-based | Local identity | Limiting |
| **Huevos Premium Cabral** | Premium positioning | Quality signal | Narrow |
| **Campo Limpio** | Clean country | Fresh image | Common |

### Recommended: **Granja Cabral**

- Personal connection to owner
- Easy to remember
- Scalable (can add product lines)
- Searchable name
- Domain available

## 2. Brand Elements

### Visual Identity

#### Logo Concepts

**Concept 1: Classic Farm**
- Silhouette of hen/egg
- Warm earth tones
- Traditional feel

**Concept 2: Modern Minimal**
- Clean lines
- Single color
- Simple icon

**Concept 3: Route 2 Story**
- Map element
- Road motif
- Regional pride

### Color Palette

| Color | Hex | Use | Meaning |
|-------|-----|-----|---------|

### Typography

| Element | Font | Style |
|---------|------|-------|
| Brand name | Serif (e.g., Playfair) | Elegant, trustworthy |
| Tagline | Sans-serif (e.g., Lato) | Clean, readable |
| Body text | Sans-serif | Modern, clear |

## 4. Brand Story

### The Granja Cabral Story

```
Granja Cabral es una granja familiar ubicada en la 
Ruta 2, dedicada a la producción de huevos frescos 
y de alta calidad.

Con [X] años de experiencia, criamos nuestras gallinas 
con cuidado y respeto, alimentándolas con los mejores 
ingredientes para producir huevos nutritivos y deliciosos.

Cada huevo que vendemos es recolectado, seleccionado y 
entregado fresco a nuestros clientes.

Nuestra promesa: frescura, calidad y el sabor del campo.
```

## 7. Brand Consistency

### Logo Usage Rules

DO:
- [ ] Use approved colors
- [ ] Maintain minimum size
- [ ] Keep clear space
- [ ] Use high resolution

DON'T:
- [ ] Stretch or distort
- [ ] Add effects
- [ ] Use unapproved colors
- [ ] Place on busy backgrounds

### Voice & Tone

| Aspect | Tone |
|--------|------|
| Professional | Friendly but not casual |
| Quality | Confident, not arrogant |
| Approachable | Warm, personal |
| Reliable | Consistent, dependable |
