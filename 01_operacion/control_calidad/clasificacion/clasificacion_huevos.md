# Egg Grading and Classification Standards

## 1. International Grading Systems

### USDA (United States) Standards
Standard system used internationally as reference.

| Grade | Quality Level | Description |
|-------|---------------|-------------|
| AA | Premium | Highest quality, thick albumen, firm yolk |
| A | High | High quality, good appearance |
| B | Standard | Acceptable quality, may have minor defects |
| C | Processing | For further processing only |

### European Union Standards
| Class | Description |
|-------|-------------|
| Class A (Extra) | Fresh eggs, highest quality |
| Class B | Second quality, for processing |

## 4. Quality Criteria

### 4.1 Exterior Quality (Shell)

| Factor | Grade A | Grade B | Grade C |
|--------|---------|---------|---------|
| Cleanliness | Clean, no stains | Slightly stained | Stained, dirty |
| Shape | Normal | Slightly abnormal | Abnormal |
| Soundness | Sound, intact | Sound, intact | May have cracks |
| Texture | Normal | Normal | Abnormal ridges |

#### Shell Defects
- [ ] Cracks: Hairline to severe
- [ ] Checks: Minor impact damage
- [ ] Slabs: Rough areas
- [ ] Body checks: Bumps/ridges
- [ ] Misshapen: Abnormal shape
- [ ] Calcium deposits: White spots
- [ ] Pimpled: Small bumps

### 4.2 Interior Quality

#### Albumen (White) Quality

| Score | Description | Haugh Units |
|-------|-------------|-------------|
| Clear | Runs freely, thin | <60 |
| Reasonably Clear | Slight turbidity | 60-72 |
| Reasonably Firm | Holds shape moderately | 72-78 |
| Firm | Holds shape well | 78-89 |
| Very Firm | Stands high and firm | 90+ |

#### Yolk Quality

| Factor | Grade A | Grade B | Grade C |
|--------|---------|---------|---------|
| Shape | Round, normal | Slightly flattened | Flattened |
| Color | Bright orange | Slightly dull | Dull |
| Position | Centered | May be off-center | Off-center |
| Outline | Narrow outline | Moderate outline | Wide outline |

### 4.3 Air Cell

| Grade | Maximum Depth |
|-------|---------------|
| AA | 1/8 inch (3mm) |
| A | 3/16 inch (5mm) |
| B | 3/8 inch (9mm) |
| C | Over 3/8 inch |

### 4.4 Blood/Meat Spots

| Grade | Tolerance |
|-------|-----------|
| A | None visible |
| B | Small spots allowed (<1/8 inch) |
| C | Spots allowed, for processing |

## 6. Grading Process

### Visual Inspection Checklist
1. [ ] Check for cracks (gently tap or candling)
2. [ ] Verify cleanliness
3. [ ] Check shell color uniformity
4. [ ] Candle for interior quality
5. [ ] Verify air cell size
6. [ ] Check for blood/meat spots
7. [ ] Verify size (weigh sample)

### Candling Procedure
1. Darken room
2. Hold egg up to light (candler)
3. Rotate slowly
4. Observe:
   - Air cell size and movement
   - Yolk shadow position
   - Albumen clarity
   - Blood spots

### Grading Line Setup
```
Receiving → Washing → Drying → Candling → Weight Grading → Color Sorting → Packing → Storage
```

- [ ] Off odor

## 9. Haugh Unit Testing

### Formula
```
Haugh Units = 100 * log10(h - 1.7w^0.37 + 7.6)
```
Where:
- h = height of albumen in mm
- w = weight of egg in grams

### Interpretation

| Haugh Units | Quality | Grade |
|-------------|---------|-------|
| >90 | Excellent | AA |
| 72-90 | High | A |
| 60-72 | Medium | B |
| <60 | Low | C |

### Simple Field Method
1. Weigh egg
2. Crack onto flat surface
3. Measure albumen height with caliper
4. Calculate Haugh units

## 11. Best Practices

### For Maximum Quality
1. Collect eggs minimum 2x daily
2. Maintain 65-75% humidity in coop
3. Handle eggs gently, no rough stacking
4. Store pointing down (large end up optional)
5. Keep temperature consistent
6. First-in-first-out inventory
7. Train all handling personnel
8. Use quality eggs for incubation
9. Reject visibly abnormal eggs
10. Maintain clean equipment

### Common Mistakes to Avoid
1. Washing eggs (removes bloom)
2. Storing near strong odors
3. Temperature fluctuations
4. Rough handling
5. Delayed cooling
6. Improper stacking
7. Using damaged eggs for sale
