## SECCIÓN: FAQ B2B

### Título:
```
Preguntas Frecuentes para Negocios
```

### FAQs:

**1. ¿Cuál es el mínimo de compra para obtener precios mayoristas?**
```
El descuento mayorista aplica desde 20 maples (600 huevos) por mes. 
Sin embargo, podemos cotizar cualquier volumen constante. 
Consultanos sin compromiso.
```

**2. ¿Emiten factura para mi negocio?**
```
Sí, emitimos factura con todos los datos de tu empresa. 
Podemos hacer facturación mensual consolidada o por 
cada entrega, según prefieras.
```

**3. ¿Puedo establecer entregas regulares sin pedir cada vez?**
```
Absolutamente. De hecho, lo recomendamos. Establecemos 
un día y hora fijos de entrega semanal (o la frecuencia 
que necesites). Podés modificar cantidades con 
24-48 horas de anticipación.
```

**4. ¿Qué pasa si necesito un pedido urgente fuera de lo programado?**
```
Entendemos las emergencias. Con gusto coordinamos 
entregas adicionales cuando sea posible. 
Los clientes Oro y Platinum tienen prioridad 
para entregas urgentes.
```

**5. ¿Los precios son fijos o varían?**
```
Intentamos mantener precios estables. Los clientes 
Platinum pueden bloquear precios por 3 meses. 
Cualquier cambio de precio se comunica con 
15 días de anticipación.
```

**6. ¿Entregan en mi zona?**
```
Atendemos Coronel Oviedo y Ruta 2 (Km 120-150). 
Para zonas fuera de este rango, consultanos - 
podemos ver opciones para pedidos grandes.
```

**7. ¿Puedo visitar la granja antes de comprometerme?**
```
¡Por supuesto! Te invitamos a conocer nuestras 
instalaciones. Verás cómo trabajamos y la calidad 
de nuestras gallinas. Coordinamos visitas de 
lunes a sábado previa cita.
```

**8. ¿Tienen certificaciones?**
```
Estamos en proceso de certificación SENACSA. 
Trabajamos bajo estándares de calidad y 
bienestar animal. Te podemos mostrar nuestros 
protocolos en una visita.
```

**9. ¿Qué métodos de pago aceptan?**
```
Aceptamos:
- Efectivo contra entrega
- Transferencia bancaria
- Giros
- (Próximamente: MercadoPago para negocios)

Los clientes con facturación mensual tienen 
plazo de pago de 7 días.
```

**10. ¿Puedo hacer un pedido de prueba antes de comprometerme?**
```
¡Totalmente recomendado! Hacé un pedido pequeño, 
probá la calidad, conocenos. Sin mínimos forzados, 
sin contratos. Solo si estás conforme, hablamos 
de un suministro regular.
```
